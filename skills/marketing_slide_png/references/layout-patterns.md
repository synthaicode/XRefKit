# Marketing Slide PNG Layout Patterns

Use these patterns as the default library for Marketing Group slide construction.

## 1. Title Compare

Use when introducing a theme through contrast.

- structure:
  - slide title
  - subtitle
  - 2 large panels
- example uses:
  - before / after
  - unstable / stable
  - naive use / designed use

## 2. Cause And Result

Use when showing why a problem appears.

- structure:
  - left: missing inputs or causes
  - center: system or AI
  - right: consequences

## 3. Sequential Flow

Use when showing ordered handoff or dependency.

- structure:
  - horizontal steps
  - chevrons between steps
  - optional summary panel under the flow

## 4. Hub And Spoke

Use when one central concept is supported by multiple surrounding elements.

- structure:
  - center hub
  - 4-6 orbit labels
  - thin connectors

## 5. Risk Contrast

Use when showing dangerous usage versus safe usage.

- structure:
  - left panel for misuse
  - right panel for controlled pattern

## 6. Role Grid

Use when listing multiple groups or responsibilities.

- structure:
  - 2x2, 3-column, or multi-row card grid
  - each card has role name and one short responsibility line

## 7. Lifecycle Loop

Use when emphasizing repeatable operational control.

- structure:
  - closed loop or repeated sequence
  - each step has one clear label

## 8. Closing Summary

Use when ending a deck.

- structure:
  - 3 columns:
    - problem
    - response
    - outcome

## Pattern Selection Rule

- If the slide compares two states, use `Title Compare` or `Risk Contrast`.
- If the slide explains causality, use `Cause And Result`.
- If the slide explains steps, use `Sequential Flow`.
- If the slide explains supporting systems, use `Hub And Spoke`.
- If the slide explains organization, use `Role Grid`.
- If the slide closes the deck, use `Closing Summary`.
